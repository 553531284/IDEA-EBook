# 👽 IDEA EBook

你甚至可以用IDEA 写代码、可以在IDEA中看电子书的摸鱼插件。

## 🔎 搜索 

搜索数据来源于[笔趣阁](http://www.xbiquge.la/)

---

## 🎫 开始阅读

选中搜索的书籍信息，单击开始阅读按钮，爬取指定数据的数据，并展示第一章节的内容

---

## 🖍 章节跳转

页面下方输入框输入整数，单击右侧跳转，即可进行指定页面跳转。上一章 下一章 按钮切换单个章节
